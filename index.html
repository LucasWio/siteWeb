UI est l’abréviation d’user interface ou interface utilisateur. L’UI design se rapporte donc à l’environnement graphique dans lequel évolue l’utilisateur d’un logiciel, d’un site web ou d’une application. l'UI fait partie de l'UX design


Le UX design consiste à penser et à concevoir un site web de manière à ce que l’expérience utilisateur soit la meilleure possible. Reste à définir ce qu’est une bonne expérience utilisateur.


Une ou un persona, du verbe latin personare (per-sonare : « parler à travers »), est une personne fictive stéréotypée



les deux types de balises

  2. Les balises dites orphelines n'encadrent aucun code ; elle se ferment elles-meme Par exemple <br/> abréviation de break row;; <hr/> abréviation de horizont row
